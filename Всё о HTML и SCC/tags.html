Html теги

Тег Пример Предназначение тега
html <html> </html> Указывает программе просмотра страниц, что это html документ
head <head> </head> Определяет место, где помещается различная информация, которая не отображается в теле документа. Здесь распологается тег названия документа и другие теги управления(подключение файлов, настройка кодировки и т.д.
pre <pre> </pre> Обормляет предварительно отформированный текст

большой

текст (абзац)
h2 <h2> </h2> Создаёт меньше чем

большого

текст

меньшего

h4 <h4> </h4> Создаёт

средний

текст
h5 <h5> </h5> Создаёт
маленький
текст
h6 <h6> </h6> Создаёт самый
маленький

b <b> </b> Создаёт жирный текст
i <i> </i> Создаёт наклонный текст текст
tt <tt> </tt> Создаёт текст имитирующий как у печатной машинки
cite <cite> </cite> Используется для цитат-обычно наклонный
em <em> </em> Используется для выделения из текста слово (жирным)
font size=? <font size=?> </font> Устанавливает текст в пределах от 1 до 7
font color=? <font color=?> </font> Устанавливает цвет текста, используя значение цвета в виде RGB
a href="URL" <a href="URL"> </a> Создаёт гиперссылку на другие документы или часть текущего документа
img src="imgURL" <img src="imgURL"> </a> Создаёт гиперссылку на рисунок, находящийся по адресу imgURL
p <p> </p> Создаёт новый параграф
p algin=? <p algin=?> </p> Выравнивает параграф относительно одной из сторон документа
br <br> </br> Вставляет перевод строки
table <table> </table> Создаёт таблицу
tr <tr> </tr> Определяет строку в таблице
td <td> </td> Определяет отдельную ячейку в таблице
th <th> </th> Определяет заголовок таблицы
